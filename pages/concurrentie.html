Concurrentie analyse

PUKKELPOP

    Positief

  1. De achtergrond is een mooi opvallend design
  2. Je ziet direct dat de grootste artiesten die komen
  3. Er is een zoekfunctie
  4. Je kan duidelijk zien dat je de site in het Engels, Frans en Nederlands kan vertalen
  5. Er is een zelfgemaakte spotify afspeellijst die op de homepage staat
  6. Waar en wanneer het festival zicht plaatsvindt staat op de homepage bij het logo

    Negatief

  1. De achtergrond maakt alles een beetje druk
  2. De navigatiebalk gaat niet mee bij het scrollen. Het blijft bovenaan de site
  3. Waar en wanneer het festival zich plaatsvindt staat er heel heel klein op

  • De navigatebalk blijft aan de linkerkant van de site
  • De site kan ik 3 talen worden vertaald
  • De data staat op de navigatiebalk

  • De navigatiebalk heeft veel opties
